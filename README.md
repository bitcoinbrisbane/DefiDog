# DeFi Dog

LeetCode problems solved in Solidity. Learn algorithms. Ship smart contracts. Stay alpha.

## About

DeFi Dog is a YouTube channel dedicated to teaching algorithmic problem-solving using Solidity. Each video breaks down a classic LeetCode problem and implements a gas-optimized solution for the EVM.

## Topics Covered

- Arrays & Strings
- Hash Tables & Mappings
- Binary Search
- Dynamic Programming
- Trees & Graphs
- Gas Optimization
- Inline Assembly
- Storage Patterns
